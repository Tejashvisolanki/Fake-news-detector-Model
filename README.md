# Fake-news-detector-Model
A supervised machine learning system that classifies news articles as real or fabricated with 99.3% F1-score, built on the ISOT Fake and Real News Dataset (44,898 articles sourced from Reuters and fact-checked outlets).
Pipeline: Raw article text is cleaned through a 7-step NLP preprocessing chain — lowercasing, URL/HTML removal, tokenization, stopword filtering, and WordNet lemmatization — then transformed into a 50,000-feature TF-IDF matrix capturing unigrams and bigrams.
Modelling: Five classifiers were benchmarked — Logistic Regression, Naive Bayes, Linear SVM, Random Forest, and XGBoost. GridSearchCV hyperparameter tuning (5-fold cross-validation) was applied to the top candidates. Tuned Logistic Regression is the optimal model: 99.3% accuracy, 99.9% ROC-AUC, sub-3-second training time, and CV standard deviation below 0.001.
Key Findings: Attribution language ("reuters", "said", "official") strongly predicts credibility. Sensationalist vocabulary ("deep state", "exposed", "breaking"), high capitalization ratios (7.1% vs 3.8%), and excessive punctuation are the dominant fake news signals. Linear models outperform tree ensembles on high-dimensional sparse text — a consistent finding in NLP literature.
Deliverables: Clean Jupyter notebook, formal project report, and an interactive HTML dashboard with a live classifier, EDA visualizations, ROC curves, and feature importance panels.
Dataset - kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset
